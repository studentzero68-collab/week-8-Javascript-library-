# JavaScript Utility Library

## Overview

This project contains a collection of reusable JavaScript utility functions created as part of the Zaio JavaScript Problem Solving, Algorithms, and Array Methods sprint.

Each function is written as an ES module, includes JSDoc comments, and contains test cases using `console.assert()`.

## Running the Files

Make sure Node.js is installed.

Run a file using:

```bash
node utils/capitalise.js
```

## Testing

Each utility file contains at least 3 test cases using `console.assert()`.

If no errors appear in the terminal, the tests have passed.
